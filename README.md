# 🌿 EcoWallet - Controle Financeiro Inteligente

> Uma carteira digital moderna, rápida e intuitiva para você gerenciar suas receitas e despesas com estilo.

## 💻 Sobre o Projeto

O **EcoWallet** é uma aplicação Full-Stack desenvolvida para facilitar o controle financeiro pessoal. Com uma interface atraente baseada em princípios de *Glassmorphism* e microinterações, o usuário tem uma experiência fluida ao adicionar entradas e saídas, vendo seu saldo atualizar em tempo real.

O projeto foi desenhado não apenas para ser funcional, mas para oferecer um design moderno e responsivo, focado na melhor experiência do usuário (UX/UI).

## ✨ Funcionalidades

- **Dashboard Financeiro:** Adição rápida de transações (Receitas e Despesas).
- **Cálculo Automático:** O saldo total é atualizado em tempo real na tela.
- **Microinterações:** Efeitos visuais (cifrões animados) que geram feedback imediato ao usuário a cada nova transação.
- **Design Moderno:** Utilização de Flexbox, Box-shadows e Glassmorphism.
- **Cartões Interativos:** Página inicial com cartões de informações ocultas (Hover effects).
- **Links Sociais 3D:** Botões com animação isométrica para LinkedIn, WhatsApp e Instagram.

## 🛠️ Tecnologias Utilizadas

### Front-end (UI/UX)
- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização avançada (Glassmorphism, Flexbox, Keyframes para animações).
- **JavaScript (Vanilla):** Lógica de cálculo, manipulação do DOM e disparos de eventos.

### Back-end & Infraestrutura (Em Desenvolvimento 🚧)
- **Python / FastAPI:** Criação da API RESTful para processamento de regras de negócio.
- **PostgreSQL:** Banco de dados relacional para persistência segura das informações.
- **Docker & Docker Compose:** Containerização de todo o ambiente de desenvolvimento.
